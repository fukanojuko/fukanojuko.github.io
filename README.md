# fukanojuko.github.io

ゲーム制作団体 **fukanojuko** のサイト。**作品・デザイン・制作メモ**を発信する。

- 公開URL: <https://fukanojuko.github.io/>
- 作品の固定URL（コンテスト提出用）: `https://fukanojuko.github.io/works/<slug>/`

## セクション

| パス | 内容 |
| --- | --- |
| `/works/<slug>/` | 作品紹介（コンテスト提出用の固定URL） |
| `/designs/<slug>/` | デザインラフ・WIP の共有 |
| `/notes/<slug>/` | 制作メモ・devlog |

## 編集・開発

- **コンテンツ編集（ブラウザのみ・非エンジニア向け）**: `/admin` から。手順は [CONTRIBUTING.md](./CONTRIBUTING.md#2-コンテンツ編集非エンジニア向け)
- **ローカル開発・構成・デプロイ・PRフロー**: [CONTRIBUTING.md](./CONTRIBUTING.md)

技術スタック: Astro 6（Content Collections）／ Sveltia CMS（Git-backed CMS）／ GitHub Actions → GitHub Pages。
